# Pulse Anomaly Playbook

Use this playbook when reviewing week-over-week movement.

- Treat a 15% or larger week-over-week move as notable.
- Treat a 25% or larger move as a headline change.
- Flow metrics use weekly sums: signups, activated, paidConversions,
  churnedAccounts.
- Stock metrics use end-of-period values: mrr, activeAccounts.
- Activation rate is activated / signups.
- Paid conversion rate is paidConversions / signups.
- Churn is a count in this demo dataset, not a churn-rate denominator.
- If multiple funnel metrics move together, describe the shared direction
  instead of inventing a causal story.
